# Agent Trajectory Prediction
Trajectory Prediction using Agroverse1 Motion Forecasting Dataset

This repo contains progressively more advanced attempts at simulating automotive agents using the 
Agroverse 1 Motion Forecasting Dataset. 

Each attempt is presented in turn. The intention is to explore grander and more advanced concepts,
maintaining a view on accuracy, inference time and additional featurs

# Data
The raw data can be downloaded from 
https://www.argoverse.org/av1.html#forecasting-link
please download and uncompress the train, val and test sets within a data folder
The resulting path should be

data\forecasting_test_v1.1\test_obs\data

# Maps
The vector maps can be downloaded from 
https://www.argoverse.org/av1.html#forecasting-link

please download and uncrompress within a maps folder. The resulting path should be
maps\hd_maps\map_files
